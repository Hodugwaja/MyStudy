식별자

문서 내에서 특정한 엘리먼트를 식별하기 위한 용도로 사용되는 API

Element.tagName

해당 엘리먼트의 태그 이름을 알아낸다

단 태그의 이름을 변경하지 못한다

  • css
  • JavaScript

Element.ClassId

각각 태그는 하나의 id만 넣을 수 있기 때문에 단 하나의 태그만 식별할 수 있는거다

  • html
  • css
  • JavaScript

Element.ClassList

해당 엘리먼트는 className에 비해 훨씬 편리한 사용성을 제공한다

  • html
  • css
  • JavaScript

Element.ClassName

여러 개의 엘리먼트를 그룹핑할 때 사용한다
