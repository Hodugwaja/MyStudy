1. BOM

웹 브라우저의 창이나 프래임을 추상화 해서 프로그래밍적으로 제어할 수 있도록 하는 수단

여기에서는 Window 객체의 사용법에 대해서 알아봄(Window 객체의 프로퍼티와 메소드를 사용법 === BOM)



2. Window 객체

전역 객체

window 객체는 식별자 window를 통해서 얻을 수 있고 생략도 가능하다

사용자와의 커뮤니케이션

HTML은 form을 통해서 사용자와 커뮤니케이션을 할 수 있는 기능을 제공

자바스크립트는 사용자와 정보를 주고 받을 수 있는 간편한 수단으로 제공

Location 객체

Location 객체는 무선의 주소와 관련된 객체로 Window 객체의 프로퍼티이다

Window의 문서 URL을 변경 할 수 있고 문서의 위치와 관련해서 다양한 정보를 얻어낼 수 있다

현재 Window URL을 알아내기
